# MCP-Style AI Research Assistant

This project implements a lightweight agent that simulates MCP-style tool orchestration.

## Features
- Tool routing (search, calculator, summarize)
- Query-based tool selection
- Document retrieval with ranking
- Structured outputs

## Example

>> what is MCP  
→ uses search

>> calculate 20 + 5 * 2  
→ uses calculator

## Run

python main.py
