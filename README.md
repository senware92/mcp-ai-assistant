# MCP-Style AI Research Assistant

Lightweight AI agent that routes user queries across multiple tools (document retrieval, computation, summarization) to generate grounded responses.

## Overview

This project implements a simple agent inspired by MCP-style architectures, where queries are dynamically routed to external tools instead of relying on a single-pass response.

## Features

- Tool-based query routing
- Document retrieval with basic ranking
- Safe mathematical computation
- Text summarization
- Modular tool system

## Architecture

User Query → Tool Router → Tool Execution → Response Assembly
